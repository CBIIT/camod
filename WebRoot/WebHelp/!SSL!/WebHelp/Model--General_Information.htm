Model--General Information

To enter or edit general information for an animal model,

 

  1. After you have logged into caMOD, the Submit and Edit Models page opens. If you click the Add New Model red hypertext link, the General Information dialog box opens for you to add information about the new model.

 

OR

 

  1. Click Model Characteristics from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have click the (model) Submit button in other caMOD viewers.

 

  1. Enter or select the appropriate information about your animal model in the text entry fields. Items with an asterisk are required fields

 

General Information Text Fields

Description

*Model descriptor

Model descriptor is the commonly used name for the model. The model descriptor is

like a "nick name" for the model (e.g. conditional BrCa1 knockout).

In some cases the model name might be the same as the genotype (e.g. Wap-Cre).

The program accepts multiple models with the same name (like Melanoma model). The

record is identified by an unique number and not by the name. The search will eventually

pull up all records that match certain criteria.

Official nomenclature

Official name(s) given to the model

Genotype

The genotype is the allelic designation (Wap-Cre x BrCa1 fl/fl x p53+/- ). In some cases the model descriptor might be the same as the genotype (e.g. Wap-Cre)

*PI Email Address

Email address of the Principal Investigator for the model. If none is available, enter None.

Is the model a toolhouse?

Answer Yes or No.

*Species

Select genus and species of the model from the drop-down list. The list contains one mouse species, Mus musculus, and two rat species, Rattus rattus and Rattus norvegicus. If you use an animal that is not

listed please contact the help desk immediately so that we can add the species.

If other strain

Specific strain information, if appropriate.

Current Genetic Background(s)

Pertinent information about the genetic background about the model; separate entries must be separated by commas.

Experimental Design

The experimental design describes how a model was generated. This is especially important

for GEM.

Example: "The 4.5 kb 5' flanking region of the C3(1) component of rat prostatein

was utilized to target expression for SV40 Tag to the prostate epithelium."

*Phenotype

The field "Phenotype" asks for a general description of the phenotype.

Example: Expression of the SV40 early region in the mammary and prostate epithelium

leads to invasive carcinoma development over a predictable time course. The transgene

is expressed in mammary ductal cells and the terminal ductal lobular unit in virgin

animals and without hormone stimulation.

*Gender

Select gender information from the drop-down menu.

Breeding Notes

Enter information about breeding problems or other issues related to the breeding

of this model, e.g. infertility in males. Information about homo- or heterozygosity

may also be stored in this field. This information is especially valuable when the

strain is kept heterozygous, but the phenotype occurs only in homozygous mice.

Website for added information

Add a website URL where more information about the experiment(s) and/or models can be obtained.

*Record Release Date

The date when the models records are to be release: Immediately, or on the date you specify in this field.

You can choose between:

Option 1: The record will be shown immediately.

Option 2: The record will be shown after a specified date. Select the date from the

calendar that pops up when you select the radio button.

We will contact the submitter of the record about a week prior to the release date

and confirm the date. If we do not receive a response from the submitter the record

will not be displayed.

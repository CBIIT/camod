Induced Mutation

The "induced mutation" page gathers information about mutations that were triggered by chemicals, radiation or other types of mutagenes and that become hereditary.

 

To enter or edit  the Targeted Modification information for an animal model:

 

  1. Click Genetic Description from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.

 

  1. Click the Enter Induced Mutation red hypertext link. This opens the Induced Mutation form.

 

  1. Enter or select the appropriate information about your animal model in the text entry fields. Items with an asterisk are required fields.

 

Induced Mutation Text Fields

 

Description

*Name of Inducing Agent

Enter the name of the agent inducing the mutation.

*Inducing Agent Category

 

Inducing agent is that which caused the mutation. Select the agent category from the drop-down list or select Other and enter an agent category not found in the drop-down list in the following text box.

Other Category

Enter the Induced Mutation Category  that is not listed in the drop-down list.

CAS Number

Enter the Chemical Abstracts Service (CAS) number for a chemical or drug. Use the provided hypertext link (Find CAS#) to find the CAS number. Copy the number and paste in into the filed on the caMOD Chemical/Drug page.

 

Note: You can also retrieve the CAS number from the DTP page described in the previous row of this table. It will display with the NSC number on the Chemical/Drug page.

Gene ID (Entrez)

Enter the Gene Entrez ID number. Click the Find ID button for a link to Entrez where you can locate the ID.

Description

 Describe in the text field how the mutation was induced.

Observation

Describe your observation about the induced mutation.

Method of Observation

Explain how you learned about the gene(s) that were modified?

 

MGI Number

Enter the Mouse Genome Informatics (MGI) ID for this mutation.  MGI Number is a hypertext link to the MGI website where you can learn more about the MGI.

Comments

Comment on the contents of this page or use this field to add data for this model. You can also enter information that you would like to share but that did not fit in any other field.

 

  1. Click the Submit button to execute the submission of the model information.

 

  1. Click Reset to return the form to default settings: If you are entering data for the first time, Reset clears the form (resets it to its defaults). If you are changing data for an existing model, Reset returns the form back to the unmodified state (the default).

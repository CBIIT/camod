Advanced Search Mode

Once on the caMOD Search page, the Search options display at the upper left of your viewer.

 

Note: When defining search criteria in caMOD, there are no "required" fields. You can add much detail, or only those parameters that you consider essential to the search.  

 

  1. From any caMOD viewer window, click the Search Models link at the top of the page. The KeyWord Search and Simple Search options display by default.

 

  1. Click Advanced Search to launch a search based on very specific and detailed search criteria.

 

  1. In the Advanced Search Mode dialog box,  enter the specific parameters for your search:

 

Search Parameters

Description

Model Name/Model Descriptor

Enter the name or parts of the model name. The search will return all models that contain the word or letter combination you entered.

Principal Investigator's (PI) Name

Select from the drop-down list the name of the Principal Investigator for the model

Site of Lesion/Tumor

Diagnosis

For Lesion/Tumor Site or Diagnosis, click the corresponding Select button. Select from the Organ/Tissue/Disease vocabulary tree that opens the  site of the corresponding lesion or tumor or disease indicated in this model.

Species

Select from the Species drop-down menu the animal model species.

Phenotype

Enter a brief description or portion thereof for the Phenotype. The search will return all models that contain the word or letter combination you entered.

 

Example: Expression of the SV40 early region in the mammary and prostate epithelium leads to invasive carcinoma development over a predictable time course. The transgene is expressed in mammary ductal cells and the terminal ductal lobular unit in virgin animals and without hormone stimulation.

Genetic Description:

  • Gene Name

  • Genomic Segment Designator

  • Inducing Agent for Induced Mutation

Check Engineered Transgene (gene was used as the transgene) or Targeted Modification (gene was knocked out) before you can enter the Gene Name in its corresponding text box. You can select both methods of genome manipulation, but you can also both boxes unchecked and retrieve all models with manipulations of this gene.

 

Enter the Genomic Segment Designator (also called the Clone Designator) which is location of the clone in the library.

 

Select from the drop-down menu the Inducing agent if there was an induced mutation.

Carcinogenic Agents:

 

 

  • Carcinogenic Interventions

  • Chemical/Drug

  • Growth Factor

  • Hormone

  • Radiation

  • Surgery

  • Virus

The carcinogenic agents are divided in different categories. A drop-down list shows items sorted by category. The lists contain only agents and treatments that were used with models in the database. Therefore, selecting one of these list items will retrieve at least one matching record.

  • Check the checkbox to search for models with Carcinogenic Interventions data. Note: You must check this box to make drop-down lists in the section available.  

  • Select a listed chemical or drug. This retrieves models that were exposed to this chemical or drug.

  • Select a listed growth factor. This retrieves models that were treated with the growth factor.

  • Select a listed hormone. This retrieves models that were exposed to this hormone.

  • Select a listed type of radiation. This retrieves models that were exposed to this type of radiation.

  • Select a listed surgery or other treatment. This retrieves models treated with selected surgery or other treatments.

  • Select a listed virus. This retrieves models that were exposed to this virus.

Cell Lines

Enter the name or parts of the name to search for a specific cell line.

Therapeutic Approaches

 

Check to search for models with specified therapeutic approaches. At that point, the Compound/Drug text box becomes enabled where you can add an appropriate name or part of a name. When you select the compound name, all models with therapeutic approaches data will be returned.

Histopathology

Check the box to search for models with metastasis.

Microarray Data

Check the box to search for models with associated microarray data.

 

  1.  Click the Search button to launch the search. To clear all field, click the Clear button.
 

Results display in the Search Results viewer.

 

Notes about Advanced Search:

 

All items in all drop-down lists should return results as terms that have been used in the description of models in the "edited-approved" state.

 

 

 

 

Combining Search Criteria:

 

You can combine search criteria with an "AND" statement.

 

Example:

 

If you launch a search for models where the model descriptor/name contains WAP, the search might return six matches.

 

If you search for models where the model descriptor/name contains WAP and the tumor site is mammary gland, the search might return less than six matches, because both search criteria need to be fulfilled in order to get matching records. This is an easy way to narrow the number of retrieved records.

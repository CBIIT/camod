Advanced Search Mode

Once on the caMOD Search page, the Search options for caMOD display at the upper left of your viewer.

 

Note: When defining search criteria in caMOD, there are no "required" fields. You can add much detail, or only those parameters that you consider essential to the search.  

 

  1. From any caMOD viewer window, click the Search Models link at the top of the page.

 

  1. Click Advanced Search to launch a search based on very specific and detailed search criteria.

 

  1. In the Advanced Search Mode dialog box,  enter the specific parameters for your search:

 

Search Parameters

Description

Model Name/Model Descriptor

Enter the name of the model.

Principal Investigator's (PI) Name

Select from the checkbox the name of the project PI.

Site of Lesion/Tumor

Diagnosis

For Lesion/Tumor Site and Diagnosis, click the corresponding Select button. Select from the Organ/Tissue/Disease vocabulary tree that opens the  site of the corresponding lesion or tumor or disease indicated in this model.

Species

Select from the Species drop-down menu the animal model species.

Phenotype

The field Phenotype asks for a general description of the phenotype, which is the manifestation of a genetic trait.

 

Example: Expression of the SV40 early region in the mammary and prostate epithelium

leads to invasive carcinoma development over a predictable time course. The transgene

is expressed in mammary ductal cells and the terminal ductal lobular unit in virgin

animals and without hormone stimulation.

Genetic Description:

  • Gene Name

  • Genomic Segment Designator

  • Inducing Agent for Induced Mutation

Enter the Gene Name and specify if the gene was used as the targeted transgene,  the transgene or was knocked out (targeted mutation). You can select all three methods of genome manipulation, but you can also leave all boxes unchecked and retrieve all models with manipulations of this gene.

 

Enter the Genomic Segment Designator (also called the Clone Designator) which is location of the clone in the library.

 

Select from the drop-down menu the inducing agent if there was an induced mutation.

Carcinogenic Agents:

 

  • Chemical/Drug

  • Growth Factor

  • Hormone

  • Radiation

  • Surgery

  • Virus

The carcinogenic agents are divided in different categories. A drop-down list shows items sorted by category. The lists contain only agents and treatments that were used with models in the database. Therefore, selecting one of these list items will retrieve at least one matching record.

  • Search for a listed chemical or drug. This will retrieve models that were exposed to this chemical or drug.

  • Search for a listed growth factor. This will retrieve models that were treated with the growth factor.

  • Search for a listed hormone. This will retrieve models that were exposed to this hormone.

  • Search for a listed type of radiation. This will retrieve models that were exposed to this type of radiation.

  • Search for a listed surgeries and other treatments.

  • Search for a listed virus. This will retrieve models that were exposed to this virus.

 

Cell Lines

Enter the name or parts of the name to search for a specific cell line and/or select the species that the cell line was derived from.

Therapeutic Approaches

 

Check to search for models with specified therapeutic approaches. At that point, the Compound/Drug text box becomes enabled where you can add an appropriate name.

Histopathology

Check the box to search for models with metastasis.

Microarray Data

Check the box to search for models with associated microarray data.

 

  1.  Click the Submit button to launch the search. Results display in the Search Results viewer.
 

 

Combining Search Criteria:

 

You can combine search criteria with an "AND" statement.

 

Example:

 

If you launch a search for models where the model descriptor/name contains WAP, the search might return six matches.

 

If you search for models where the model descriptor/name contains WAP and the tumor site is mammary gland, the search might return less than six matches, because both search criteria need to be fulfilled in order to get matching records. This is an easy way to narrow the number of retrieved records.

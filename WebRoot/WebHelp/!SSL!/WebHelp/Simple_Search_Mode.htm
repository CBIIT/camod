Simple Search Mode

 

Note: When defining search criteria in caMOD, there are no "required" fields. You can add much detail, or only those parameters that you consider essential to the search.  

 

  1. From any caMOD viewer window, click the Search Models link at the top of the page. The Keyword Search and Simple Search options display by default.

 

You can launch a search using either the Keyword Search or Simple Search options. Additionally, once you are on the caMOD search page, all Search options display on the left sidebar of your viewer.

 

  1. For the Keyword Search, enter the query keywords. Note: You cannot combine search terms by an AND statement in the Keyword Search feature.

 

  1. In the Simple Search form, enter search parameters in one or more fields. If you enter criteria in more than one of the four fields, they will be connected by an AND statement.

 

Search Criteria

Description

Keyword Search

Enter one or more keywords to be used as the query term(s) for the search.

Model Name/Model Descriptor

Enter the name or parts of the model name. The search will return all models that contain the word or letter combination you entered.

Principal Investigator's Name

Select from the drop down list the name of the Principal Investigator for the model

Site of the Lesion/Tumor

Click the Select button. This links you to the EVS tree where you can launch a search in the EVS data tree for models that develop lesions or tumors in specific organs.

Species

Select the species of the model from the drop-down list.

 

 
  1. To launch the search, click the Search button. To clear all fields, click the Clear button.
 

Results display in the Search Results viewer.

 

 

Combining Search Criteria:

 

You can combine search criteria with an "AND" statement.

 

Example:

 

If you launch a search for models where the model descriptor/name contains WAP, the search might return six matches.

 

If you search for models where the model descriptor/name contains WAP and the tumor site is mammary gland, the search might return less than six matches, because both search criteria need to be fulfilled in order to get matching records. This is an easy way to narrow the number of retrieved records.

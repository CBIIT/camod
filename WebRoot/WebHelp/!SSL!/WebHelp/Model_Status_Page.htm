Model Status Page

 

After you have clicked the Submit button on any of the pages where you entered information for a new model, the Model Status page displays.  The default status is Incomplete, although the text does indicate that you have successfully created a new model. In the Incomplete state, the model cannot be viewed by the public. Click the Set Model Status to Complete button to finalize the submission of the model.

 

Back in the Submit and Edit Models page, the status of each model in the database is indicated:

 

Examples:

  • Edited -- Approved

  • Screen -- Assigned

  • Complete -- Not screened

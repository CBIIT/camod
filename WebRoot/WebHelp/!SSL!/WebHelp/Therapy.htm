Therapy

To enter or edit  the carcinogenic interventions  for an animal model:

 

  1. Click Therapeutic Approaches from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.

 

  1. Click the Enter Therapy red hypertext link. This opens the Therapy form.

 

  1. Enter or select the appropriate information about your animal model in the text entry fields. Items with an asterisk are required fields.

 

Therapy Fields

Description

*Drug/Compound Name

Enter the name of the drug / compound (chemical name preferred over brand name) that

was administered to the animal. If a drug cocktail consisting of several compounds

was given enter each compound separately and explain the experiment in more details

in the Experiment or the Comment field.  

NSC Number

Enter the NSC number in the text box. The NSC number is a unique drug identifier issued by the Nomenclature Standards Committee (NSC) of the U.S Food and Drug Administration.

 

If you don't know the number, click the Find NSC# button. The button opens a page at the Developmental therapeutics Program (DTP) at NCI which allows you to find the NSC number for a specific drug.

  • On the DTP page, select Chemical name starts or Chemical name number. (By default, the radio button is set to find the NSC number.)

  • Enter text in the Search Parameters field, and click Submit.

  • Once you have found the NSC number, copy the number from the field onto the field on the caMOD Chemical/Drug page.

CAS Number

Enter the Chemical Abstracts Service (CAS) number for a chemical or drug. Use the provided hypertext link (Find CAS#) to find the CAS number. Copy the number and paste in into the filed on the caMOD Chemical/Drug page.

 

Note: You can also retrieve the CAS number from the DTP page described in the previous row of this table. It will display with the NSC number on the Chemical/Drug page.

Chemical Class

Select the chemical class from the displayed list; make sure it is highlighted, and move it down to the lower box using the "" arrow.

 

To deselect a chemical class, highlight it in the right box. Move it up to the upper box by clicking the " "arrow.

 

Note: You can move multiple items at one time. Use the convention select methods: CTRL + click to select non-contiguous items; SHIFT + click to select all items within a range.

 

Biological Process

Select the biological processes affected by the drug from the displayed list. Using the up and down arrows as described above, create a list of biological processes affected by the selected drug.

Target

Using the right and left arrows as described above, create a list of targets used in the therapy.

Toxicity Grade

Indicate the toxicity grade, as observed in this experiment,  from the drop-down list: Mild, Moderate or Severe

Dose (number only)

Enter the dose of the therapeutic agent. Select the dosage units from the drop-down menu.

Gender

Select gender from the drop-down list.

Age

Enter animal age information in the text box and from the drop-down list.

Administration Routes

Select the course in which a drug was administered in order to reach the site of action in the body.

Biomarker

Biological markers (Biomarkers) are objectively measurable phenotypic parameters (for example, specific enzyme concentration, specific hormone concentration, specific gene phenotype, presence of biological substances) that characterize an organism's state of health or disease, likelihood of developing a disease, or response to a particular therapeutic intervention. Indicate the biomarkers (if any) that were used in this experiment.

Tumor Progression

Enter the time to pre-neoplastic lesion malignancy metastasis in the text box and from the drop-down list.

Experiment

Enter the hypothesis for the experiment conducted with this model and this drug.

Results

Describe results of the experiment that was conducted with this model and this drug.

Comments

Comment on the contents of this page or use this field to add data for this model. You can also enter information that you would like to share but that did not fit in any other field.

 

  1. Click the Submit button to execute the submission of the model information. Click Reset to clear the form.

 

Note: After the data have been submitted, you are offered a link to the Publications page to enter associated publications in which your specific therapy was described. By entering the publication in this way, an association between the publication and the therapy is established, and this particular publication will be displayed on the therapy search page. In addition, the publication will display on the publication search page.

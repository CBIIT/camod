Admin Functions Overview

caMOD Admin functions include responsibilities for three assignees:

 

  • Coordinator

 

The coordinator:

- Can add new users

- Appoints screeners

- Appoints editors

- Assigns models to a screener

- Assigns models to an editor

 

  • Screener

The screener scans submitted, completed records for appropriateness. The screener can approve or reject a record.

 

  • Editor

The editor checks the record for completeness and correctness. The editor approves the record for display or asks the submitter for more information.

 

Additionally, users of caMOD can fall into one of two categories:

 

  • Guest User

The guest User is any user without a valid system username and password. Guest users can search the cancer models database for public data.

 

  • Registered User

The Registered User is any user with a valid system username and password.

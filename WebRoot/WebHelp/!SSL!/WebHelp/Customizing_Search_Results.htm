Customizing Search Results

To customize search results display,:

 

  1. Click the Customize Results hypertext link in the lower left sidebar. (This link displays in the left sidebar in any caMOD viewer window.)

 

  1. The dialog box that opens displays Available Columns and Columns to Display.  In either select box, select fields and move them to the other box using the Forward () or Back () buttons . A maximum of five columns can display at one time in a Search Results viewer.

 

  1. You can also select the number of hits that display on each page. Select the number from the drop-down menu at the bottom of the page.

 

  1. Click the Save button to save your settings. Any searches you launch after this configuration will display the fields and number of hits you defined in the Search Results page.

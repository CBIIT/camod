Microarray Data

 

Microarray data that is submitted through caMOD is stored in the caArray database. You cannot work with that data in caMOD, but you must log into caArray and work with the data there. You must have a user account in caArray that is separate from your caMOD user account. Refer to: http://caarray.nci.nih.gov/

 

To enter microarray data relevant to an animal model:

 

  1. Click Microarray Data from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have click the (model) Submit button in other caMOD viewers.

 

  1. Click the Enter Microarray Data red hypertext link.

 

  1.  The dialog box that opens describes the required steps to uplad microarray data to caArray.

 

Note:  You can submit microarray data only in the Affymetrix format, in GenePix formats or in the Gene Array List.

 

 
For more information, refer to the caArray online help or the caArray 1.3 User's Guide.

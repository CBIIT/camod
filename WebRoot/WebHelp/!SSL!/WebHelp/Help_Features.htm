Help Features

Online Help in caMOD is accessible with several methods:

 

  • Quick Help Option 1 - Move your mouse pointer over the Help icon. A message appears in the bottom part of your browser.

 

  • Quick Help Option 2 - Move your mouse pointer over the Help icon and hold it there for a second or two. A tool tip message appears next to your mouse pointer.
     

  • Opening Online Help - Click on the help icon and the online help file opens in a new browser window. To access Help topics, you can use the Table of Contents, the online help index or perform a word search.

 

Note: You must close the Help window to continue the submission process.

 

 

For any general information about the application, application support or to report a bug, contact NCICB Application Support.

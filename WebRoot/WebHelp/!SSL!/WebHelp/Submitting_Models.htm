Submitting Models

After you have logged into caMOD, the Submit and Edit Models page opens. If you click the Add New Model red hypertext link, the Model Characteristics dialog box opens for you to add information about the new model.

 

Notes about Submitting Models:

 

  • You must enter Model Characteristics about your model.

 

  • You need to fill out only the parts that apply to your model, such as if your model was not exposed to carcinogenic interventions you don't have to enter data about carcinogenic interventions.

 

  • You can make multiple entries for each category, such as if the model was exposed to two different chemicals or if the model was used in multiple therapeutical trials. Once an initial entry has been made, submit it. Enter multiple entries for the same model in exactly the same manner. See the Generic Steps for Submitting Models below. A newly-named link will display in the same options list as described in the steps below.

 

  • If you cannot complete the submission of the model in one session, you can come back later and add data. caMOD will provide you with a list of all models that you entered after you have logged in.

 

 
Generic steps for Submitting Models:
 

  1. Click any category from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have click the (model) Submit button in other caMOD viewers.

 

  1. Click any red hypertext link listed under the top-level category headings. This opens the appropriate form for the data you want to enter about your model.

 

  1. Enter or select the relevant information about your animal model in the text entry fields. Items with an asterisk are required fields.

 

Once submitted, a screen opens verifying you have created a new model. By default, the status is set at incomplete; the model in this state cannot be viewed publicly.

 

  1. Click the Set Model Status to Complete button.

 

At that point, the application returns to the Submit and Edit screen where the name of the new model is listed in blue under the category for the information you added. The name is a hyperlink to the form containing the corresponding information for the model.

 

  1. To review or edit the information, click on the name of the new model.

 

 If you click the model name in the options list, the page where you have entered information displays. You can review the information or  edit it, where appropriate.

 

  1. In the same manner, enter and submit a second entry for the same model.

 
 

 

For detailed directions about submitting model information, see the specific topics listed in the online help Table of Contents.

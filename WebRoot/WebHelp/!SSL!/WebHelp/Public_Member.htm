Public Member

 

Users of caMOD can fall into one of two categories:

 

  • Guest User

The guest User is any user without a valid system username and password. Guest users can search the cancer models database for public data. See caMOD Database Search Overview for more information.

 

  • Registered User

The Registered User is any user with a valid system username and password. This user is also called a Public Member: a registered user with no administrative functions.

 

Any interested person can become a public member of caMOD, which is a Registered User.

 

To become a public member, you must enter personal information such as name and email address before you can submit a record to the database or a caMOD Coordinator must register you as a new user. See Creating a New User Account and Adding a New User. Once you submit a record to caMOD, you automatically a "public member" of caMOD.

 

  • Submitting to the database:

Public members have the right to submit records, and to edit their own records.

 

  • Querying the database:

When a public member queries the database the database will return records that the submitter defined as "public".

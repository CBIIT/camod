Genomic Segment

To enter or edit  the genomic segment information for an animal model:

 

  1. Click Genetic Description from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.

 

  1. Click the Enter Genomic Segment  red hypertext link. This opens the Genomic Segment form.

 

  1. Enter or select the appropriate information about your animal model in the text entry fields. Items with an asterisk are required fields.

 

Genomic Segment Text Fields

Description

*Integration

 

Location of Integration

Select the Random or Targeted radio button, indicating whether the integration of the genomic segment was random or targeted

.

If you selected Targeted [Integration], enter the location of integration of the genomic segment into the genome (i.e. which chromosome).

 

*Segment Type

 

  • Select the Segment Type from the drop-down list: Cosmid, YAC, BAC or Plasmid;

  • If the appropriate Segment Type is not listed, select Other from the drop-down list and enter the Other Segment Type in the text field under the list.

Other Segment Type

Enter Segment Type  that is not listed in the drop-down list.

Segment Size

Enter appropriate segment size information.

Designator

Clone Designator: Indicate the location of the clone in the library, for example, RPCI Human Male Library, 24F11.

MGI Number

Enter the Mouse Genome Informatics (MGI) ID for this mutation. MGI Number is a hypertext link to the MGI website where you can learn more about the MGI.

Comments

Comment on the contents of this page or use this field to add data for this model. You can also enter information that you would like to share but that did not fit in any other field.

 

  1. Under the gray line in the Genomic Segment  form, select or enter the requested information about the model construct, if you decide to upload a map of the construct. For more information, see Model Construct.

 

  1. Click the Submit button to execute the submission of the model information. Click Reset to clear the form.

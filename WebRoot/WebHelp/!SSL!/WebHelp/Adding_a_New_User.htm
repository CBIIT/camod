Adding a New User

 

When, as the Administrator, you want to add a new user to caMOD, follow these steps:

 

  1. Click the Admin tab in the caMOD viewer.

 

  1. Select the User Management option in the menu on the upper left of the viewer.

 

  1. In the Add User dialog box, enter the appropriate information for the new user.

 

  1. Check the Principal Investigator box if the new user is a project PI. After doing this, the PI's name will appear in any PI drop-down list in the application.

 

  1. Click the Submit button to execute the submission of the new userl information. Click Cancel.

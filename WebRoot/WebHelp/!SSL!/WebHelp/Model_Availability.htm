Model Availability

Model Availability lists sources where one can obtain or purchase animal models. Model Availability displays in caMOD under two conditions:

 

  1. If you open a model whose availability is part of the submitted model data, the Model Availability information displays underneath the Model Characteristics page. Click on the model link to display the availability details.

 

    • If the stock # is linked to the distributor's corresponding listing, that information opens.

 

    • If the stock # is not available, the stock # field is blank, the PI for the model displays as well as an active link for the distributors main web page.

 

OR

 

  1. You can open the Model Availability page where you can provide information about the availability of your model.

 

To enter model availability information for an animal model:

 

  1. Click Model Availability from the list of Submit and Edit Models menu options on the left side of the caMOD viewing window. These menu options only display after you have clicked the (model) Submit button in other caMOD viewers.

 

  1. Click any of the Model Availability options applicable to your animal model. They are listed as red hypertext links. When you click the links, the corresponding pages open in new browser windows.

 

  1. In the relevant page, enter the Strain Name and Stock # for your animal model.  Click the Find Info button to link you to the appropriate resource for that page (such as the Jackson Laboratory website) where you can look up the appropriate stock number for the model.  

 

Model Availability options are summarized in the following table:

 

Sources for animal models

Description

Jackson Lab

Jackson Lab is the largest international animal models distributor.

 

If the model is available from the Jackson Laboratory, enter the stock number of the model in the appropriate text field.

MMHCC Repository

 The Mouse Models of Human Cancers Consortium (MMHCC) Repository is an NCI-funded resource for mouse cancer models and associated straing. The repository makes strains available to all members of the scientific community. Enter the Strain Name and Stock# to inquire about the availabiltiy of the model at this site.

Investigator

 If the Investigator is willing to send the mice to other laboratories, enter the Strain Name and Stock # . Additionally, enter the Investigator's email address or the email address of the person who should be contacted regarding the mouse shipment.

IMSR

 IMSR (International Mutant Strain Resource) is a massive database linking animal model databases throughout the world. In a sense, IMSR is a "one-stop shop" for obtaining animal models. Enter the Strain and Stock# in the text fields.

 

  1. Click the Submit button to execute the submission of the model information.

 

  1. Click Reset to return the form to default settings: If you are entering data for the first time, Reset clears the form (resets it to its defaults). If you are changing data for an existing model, Reset returns the form back to the unmodified state (the default).

 

Note: If a model is a product of a crossing of multiple models, and these models are available from one or more of these sources, enter the appropriate information about these models.

 

For example: A model that is entered in the database is a conditional Brca1 knockout. To generate this knockout, two different strains were crossed: the mouse strain that carried the floxed Brca1 gene and the cre-expressing mouse strain. If the mouse strain with the floxed Brca1 gene and/or the cre-expression mouse strain are available from one or more sources, this information would be entered.
